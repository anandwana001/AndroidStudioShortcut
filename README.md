## Macros
When you require to perform repetative actions and to save your time you can create a macro of all the actions. Using a single custom shortcut can trigger this macro and this will perform all your actions for you.

To create macro you have these few steps:
 1. Record a macro
    * Open the Edit menu, point to Macros, and click Start Macro Recording.
    * Perform necessary actions that you want to be recorded.
    * Open the Edit menu, point to Macros, and click Stop Macro Recording.
    * In the Enter Macro Name dialog, specify the name for the new macro, and click OK. If the macro is intended for temporary use only, you can leave the name blank.

 2. Play back a macro
    * To play back a temporary macro, open the Edit menu, point to Macros, and click Play Back Last Macro.
    * To play back a named macro, open the Edit menu, point to Macros, and click the necessary macro name.

 3. Edit macros
    * Open the Edit menu, point to Macros, and click Edit Macros.

 4. Bind a keyboard shortcut to a macro
    * Open Settings / Preferences (âŒ˜,) and click Keymap.
    * Create a new keymap or select a keymap from the list of keymaps.
    * Expand the Macros node and select the macro for which you want to create a keyboard shortcut.
    * Right-click the macro and choose Add Keyboard Shortcut in the context menu.
    * In the Enter Keyboard Shortcut dialog, press the key combination to be used as a shortcut. The keystrokes are immediately reflected in the First Stroke field. Optionally, select the Second stroke checkbox and specify the second stroke. As you press the keys, the Preview field displays the keystrokes you pressed, and the Conflicts field displays warnings, if the keystrokes are already in use.
    * Note: use the mouse pointer, because any keystroke is interpreted as a shortcut.
    * Click OK using the mouse pointer to create a shortcut and bind it with the macro.
    * Apply changes.
